THE SOLAR SYSTEM
Be like the sun, so bright and fun
Don't be like mercury, with flames and misery
Be like venus, so close and precious
Don't be like earth, for guiding a wrong path
Be like mars, supportive and not worse
Dont be like jupiter, so angry as a fighter
Be like saturn, beautiful like a pattern
Don't be like uranus, so cold and dangerous
Be like neptune, for spreading good fortune
Don't be like pluto, so lost as far it go
Be a star, cause that's who you are.
